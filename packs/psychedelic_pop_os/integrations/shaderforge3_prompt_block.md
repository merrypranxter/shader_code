# ShaderForge3 prompt block — Psychedelic Pop OS

Style intent:
- Bubblegum neon, posterized gradients, kaleidoscopic symmetry, playful but cinematic.
- Strong color rhythm: magenta/cyan/lime/violet.
- Keep silhouettes readable; favor hypnotic motion over noisy chaos.

Uniform contract:
- u_resolution (vec2), u_time (float), u_mouse (vec2)
- u_intensity (float), u_glow (float), u_distort (float), optional u_glitch (float)

Generation hint:
“Create GLSL fragment and optional vertex shaders in a psychedelic pop style with clean composition,
high chroma color ramps, rhythmic pattern loops, and soft glow accents. Keep uniforms ShaderForge3-compatible.”
